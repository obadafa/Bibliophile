# Bibliophile
This is a website that allows users to access and read books.

## Prerequisites
1. Download Flask, write this command in terminal:
```bash 
.venv\Scripts\activate
pip install Flask
```
2. Start the server:
   
```bash
python -m flask --debug run
```
